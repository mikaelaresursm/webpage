site inspiration

When creating this website I wanted to explore the use of grids and layouts to structure my website. At first I just had some grids laid out but I wanted it to be more interactive for the user, so I took to Youtube and found that there was an Javascript element called Panzoom which allowed the user to drag the site in order to view the content. I got the code from a video called "Projects Showcase With Draggable Canvas | Panzoom Javascript" by Codegrid. The gif is an example of the function in action.
